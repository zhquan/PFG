Pagina de un restaurante

Una página web de un restaurante donde se podrá hacer reservas, pedidos(mesa o para llevar), ver si hay mesas libres.
Los clientes podrán comentar, puntuar los platos.
Un mapa para ver donde esta el restaurante y donde está el usuario para indicarle como ir desde su ubuicación.
Ranking de los platos más vendidos y de los platos mejor puntuados.

Dos modos:
1) Modo administrado:
    Donde se puede crear, subir, modificar y borrar los platos.
    Indicar las mesas ocupadas.

2) Modo usuario:
    Sólo se puede comentar(con fecha), puntuar, reservar, pedir.
